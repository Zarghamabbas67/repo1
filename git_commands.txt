Step 1 : Add your github email and username to git
  git config --global user.email "yourGitHub@email.com"
  git config --global user.name "yourGitHubusername"


Step 2 : Add file/folders to git - tracking

Step 3 : Commands
   - terminal - goto the location of the folder/project
          - git init
   - git status
   - git add
   - git commit -m “…..”
   - git remote add origin “location of remote repo”
   - git push -u origin master
   - git log
   - git --help
------------------------------------
Step 1 : Create branch 
   git branch “branch name”

Step 2 : Checkout branch
   git checkout “branch name”

Step 3 : Merge new branch in master branch
   git merge “branch name”

Step 4 : Delete branch
   git branch -d “branch name”    — delete from local
   git push origin —delete “branch name”   — delete from remote
